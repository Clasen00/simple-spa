# Пример простого spa приложения #

## Как запустить проект ##

Для запуска приложения необходимо выполнить следующие команды:

### Устанавливаем пакеты composer ###

    composer install

### Устанавливаем зависимости yarn ###

    yarn install

### Запускаем сборщик ###

    yarn dev

### Запускаем symfony ###

    php bin/console server:run
